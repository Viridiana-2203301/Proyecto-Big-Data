# Proyecto: Clasificación de Notas Periodísticas con Teachable Machine


# Descripción General

Este proyecto consiste en una aplicación web capaz de clasificar imágenes periodísticas en siete categorías informativas utilizando un modelo entrenado con Teachable Machine. El objetivo es automatizar la organización de material visual utilizado en la redacción del medio Diario Tijuana.

El sistema permite cargar una imagen, analizarla mediante el modelo de inteligencia artificial y mostrar la categoría con mayor probabilidad.

# Características del Proyecto

* Clasificación automática de imágenes.

* Modelo entrenado con Teachable Machine (Image Classification).

- Integración del modelo en una página web usando HTML, CSS y JavaScript.

- Siete clases informativas: Seguridad, Estado, Local, Clima, Educación, Política y Cultura.
